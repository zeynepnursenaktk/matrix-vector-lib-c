# # Matrix & Vector Library (C)

C dili ile geliştirilmiş matris ve vektör işlemleri kütüphanesi.
Temel matris işlemleri ve istatistiksel hesaplamalar içerir.
Akademik amaçlı olarak geliştirilmiştir.

---

## Kullanılan Teknolojiler
- C

---

## İçerik
- Matris ve vektör oluşturma
- Ortalama, standart sapma, kovaryans, korelasyon
- Matris çarpımı ve transpozu
- 3x3 matris determinantı
- Rastgele matris ve vektör üretimi
